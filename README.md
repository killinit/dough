# Frontend Assets

The Money Advice Service's frontend assets.


## Prerequisites

* [Git]
* [Bower]


## Installation

Clone the repository:

```sh
$ git clone https://github.com/moneyadviceservice/frontend-assets.git
```

Make sure all dependencies are available:

```sh
$ bower install
```
